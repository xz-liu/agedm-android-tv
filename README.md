# AGE DM TV

面向 Google TV / Android TV 的 AGE DM 原生客户端。

## 下载

| 版本 | 链接 |
|------|------|
| Release（推荐） | [agedm-tv-release.apk](release-assets/agedm-tv-release.apk) |
| Debug | [agedm-tv-debug.apk](release-assets/agedm-tv-debug.apk) |

最低系统版本：Android 8.0

## 功能

### 浏览与搜索
- 首页、推荐、更新、排行、目录、历史记录六大分区，遥控器方向键直接切换
- 目录支持按地区、类型、标签、年份、排序筛选
- 全局搜索，支持关键词模糊匹配

### 播放器
- 左右方向键快进 / 快退 10 秒，长按加速
- 调速：0.5× / 0.75× / 1× / 1.25× / 1.5× / 2×，倍速设置全局记忆，下次打开自动恢复
- 多播放源切换，某个源失败自动尝试其他源
- 选集面板，支持直接跳集
- 自动播放下一集

### 观看记录
- 自动记录每部动画的观看进度（集数 + 时间点）
- 历史记录分区展示所有看过的动画
- 主页顶部「继续观看」一键跳回上次看到的位置，打开时询问是否从断点续播

### 手机扫码投送
- 在电视上打开扫码页面，手机与电视连接同一 Wi-Fi 后扫码
- 在手机页面粘贴任意 AGE DM 链接（详情页 / 播放页）
- 电视自动跳转到对应动画或直接开始播放

## 安装

直接将 APK 拷到电视安装，或通过 adb：

```bash
adb connect <tv-ip>:5555
adb install -r agedm-tv-release.apk
```
