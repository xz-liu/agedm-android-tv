# AGE DM TV

面向 Google TV / Android TV 的 AGE DM 原生客户端。

## 下载

| 版本 | 链接 |
|------|------|
| Release（推荐） | [agedm-tv-release.apk](release-assets/agedm-tv-release.apk) |
| Debug | [agedm-tv-debug.apk](release-assets/agedm-tv-debug.apk) |

最低系统版本：Android 8.0

## 功能

### 浏览与搜索
- 扫码搜索、首页、目录、推荐、更新、排行、记录七个分区，遥控器方向键直接切换
- 首页提供今日更新、最近更新、每日推荐与每日追番分区
- 目录支持按地区、版本、类型、年份、排序筛选
- 排行榜支持按年份筛选
- 全局搜索，支持关键词模糊匹配
- 针对电视遥控器优化焦点逻辑，边界不会异常跳回导航栏

### 播放器
- 左右方向键快进 / 快退 10 秒
- 调速：1× / 1.25× / 1.5× / 2×，倍速设置全局记忆
- 多播放源切换，某个源解析或播放失败时自动尝试后续源
- 自动播放下一集时沿用当前成功播放的源
- 记住上次使用的播放源、集数和播放进度，下次进入可继续播放
- 选源、选集、继续观看都可直接在 TV 端完成

### 观看记录
- 自动记录每部动画的观看进度（集数 + 时间点）
- 记录分区展示最近观看与续播历史
- 详情页可直接从上次播放位置继续观看

### 手机扫码投送
- 扫码页作为主界面标签页直接展示，手机与电视连接同一 Wi-Fi 后即可使用
- 手机端既可以搜索动画名称，也可以提交 AGE 动漫详情页、播放页链接或动画 ID
- 手机连续选择多个动画时，电视会及时切换到最新投送内容

### 详情页
- 展示动画简介、播放源、选集、系列作品与相关推荐
- 相关推荐对齐桌面站数据，支持显示更多推荐项

## 安装

直接将 APK 拷到电视安装，或通过 adb：

```bash
adb connect <tv-ip>:5555
adb install -r agedm-tv-release.apk
```
